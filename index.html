As API's fornecem a troca de informações entre sistemas.

API PCO Aprovação pelo gestor do CC


# Exemplo curl \
-X POST http://solicitarurl.com/rest/APIPCO/APROVCC \
--header 'Content-Type: application/json' \
--data '{
    "xFilial":"301",
    "cConta":"510105002",
    "cCCusto":"1080101",
    "dData":"30/03/2024",
    "cPedido":"018199",
    "cUsrLib":"000001",
    "nValor": 100.00,
    "cSolPor":"XXXXXXX"
}'
        	

Exemplo retorno:

Com sucesso:

{
    "status": true,
    "msg": "integrado com sucesso."
}


Sem sucesso:

{
    "status": false,
    "msg": "erro."
}

BODY PARAMETERS

Campo Tipo Descrição
xFilial String (max 3) Filial
cConta String (max 12) Conta Contabil
cCCusto String (max 9) Centro de Custo
dData String (max 10)
cPedido String (max 6) Pedido de Compras
cUsrLib String (max 6) codigo do usuario Protheus
nValor Number (max 16,2)
cSolPor String (max 150) Codigo da solicitacao portal
